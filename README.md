# Overview

A web application to automatically moderate discussions.

# Setup

## Requirements

* Python 3.5+

## Installation

Install from source:

```sh
$ git clone https://github.com/dwelling-place/dwelling-place.git
$ cd dwelling-place
$ make install
```

# Usage

Run the application:

```sh
$ make run
```
